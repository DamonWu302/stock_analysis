# 回测指标与阈值清单

本文档整理当前回测系统中已接入的指标、默认阈值，以及适合做自动化参数搜索的建议范围。

说明：
- “默认值”是当前系统实际生效的默认配置。
- “建议搜索范围”用于后续脚本做参数寻优。
- 建议优先做“小范围、分组、分阶段”搜索，不要一开始全量网格爆搜。

## 1. 市场环境过滤

### 1.1 市场平均分过滤

- 参数：`market_score_filter_min_avg`
- 含义：全市场 `daily_score` 当日平均分必须不低于该值
- 默认值：`35`
- 建议搜索范围：`32 ~ 45`
- 建议步长：`1`

### 1.2 市场 5 日均值过滤

- 参数：`market_score_filter_min_ma5`
- 含义：全市场平均分的 `5` 日均值必须不低于该值
- 默认值：`35`
- 建议搜索范围：`32 ~ 45`
- 建议步长：`1`

### 1.3 大盘站上 MA20

- 参数：`market_require_benchmark_above_ma20`
- 含义：大盘收盘价必须站上 `MA20`
- 默认值：`True`
- 建议搜索范围：`True / False`

### 1.4 大盘 MA20 方向向上

- 参数：`market_require_benchmark_ma20_up`
- 含义：是否要求大盘 `MA20` 相比前一日继续上行
- 默认值：`False`
- 建议搜索范围：`True / False`

相关指标：
- 市场平均分
- 市场平均分 `5` 日均值
- 大盘收盘价
- 大盘 `MA20`
- 大盘 `MA20 / 前一日 MA20`

## 2. 组合与仓位控制

### 2.1 最大持仓数

- 参数：`max_positions`
- 默认值：`4`
- 含义：组合最多同时持有的股票数量
- 建议搜索范围：`2 ~ 6`
- 建议步长：`1`

### 2.2 单票仓位上限

- 参数：`max_single_position`
- 默认值：`0.30`
- 含义：单只股票仓位上限为组合总资产的 `30%`
- 建议搜索范围：`0.15 ~ 0.40`
- 建议步长：`0.05`

### 2.3 其他组合限制

- `allow_pyramiding = False`
  - 不加仓
- `allow_same_day_repeat_trade = False`
  - 不允许同日重复买卖同一只

执行方式：
- `T日收盘后` 生成信号
- `T+1日开盘` 执行成交
- 新开仓按“剩余现金 / 新开仓数量”等权分配
- 同时受单票仓位上限约束

## 3. 买入规则

### 3.1 严格买入 `buy_strict`

触发条件：
- `score_total >= 74`
- `score_ma_trend >= 14` 或 `score_breakout >= 12`
- `score_volume_pattern >= 14`
- `score_capital_sector >= 10`

对应可调阈值：

- 参数：`buy_strict_score_total`
  - 默认值：`74`
  - 建议搜索范围：`68 ~ 82`
  - 建议步长：`2`

- 参数：`buy_strict_score_ma_trend`
  - 默认值：`14`
  - 建议搜索范围：`10 ~ 18`
  - 建议步长：`1`

- 参数：`buy_strict_score_breakout`
  - 默认值：`12`
  - 建议搜索范围：`8 ~ 16`
  - 建议步长：`1`

- 参数：`buy_strict_score_volume_pattern`
  - 默认值：`14`
  - 建议搜索范围：`10 ~ 15`
  - 建议步长：`1`

- 参数：`buy_strict_score_capital_sector`
  - 默认值：`10`
  - 建议搜索范围：`8 ~ 16`
  - 建议步长：`1`

### 3.2 增强买入 `buy_momentum`

触发条件：
- `score_total >= 68`
- `score_volume_pattern >= 14`

对应可调阈值：

- 参数：`buy_momentum_score_total`
  - 默认值：`68`
  - 建议搜索范围：`62 ~ 78`
  - 建议步长：`2`

- 参数：`buy_momentum_score_volume_pattern`
  - 默认值：`14`
  - 建议搜索范围：`10 ~ 15`
  - 建议步长：`1`

## 4. 买入前置过滤

以下条件对两档买入都生效。

### 4.1 核心命中数

- 参数：`buy_min_core_hits`
- 默认值：`4`
- 含义：六大评分项里至少命中 `4` 项核心指标
- 建议搜索范围：`2 ~ 5`
- 建议步长：`1`

### 4.2 低位限制

- 参数：`buy_low_position_high_ratio_max`
  - 默认值：`0.95`
  - 含义：`close <= prior_20_high * ratio`
  - 建议搜索范围：`0.90 ~ 1.00`
  - 建议步长：`0.01`

- 参数：`buy_20d_gain_max`
  - 默认值：`15`
  - 含义：若不满足前一条，则允许 `20日涨幅 <= 15%`
  - 建议搜索范围：`8 ~ 25`
  - 建议步长：`1` 或 `2`

### 4.3 近 3 日放量滞涨过滤

- 参数：`buy_recent_stall_lookback`
  - 默认值：`3`
  - 建议搜索范围：`2 ~ 5`
  - 建议步长：`1`

- 参数：`buy_recent_stall_pct_max`
  - 默认值：`1`
  - 含义：涨幅低于该值且放量则视为滞涨
  - 建议搜索范围：`0.5 ~ 2.0`
  - 建议步长：`0.25`

- 参数：`buy_recent_stall_volume_multiple`
  - 默认值：`1.2`
  - 含义：成交量高于 `vol_ma5 * 倍数`
  - 建议搜索范围：`1.0 ~ 1.8`
  - 建议步长：`0.1`

### 4.4 风险限制

- 参数：`buy_risk_amplitude_max`
  - 默认值：`0.20`
  - 含义：买入前 `5` 日振幅上限
  - 建议搜索范围：`0.12 ~ 0.28`
  - 建议步长：`0.02`

- 参数：`buy_risk_max_drop_max`
  - 默认值：`0.12`
  - 含义：买入前 `5` 日最大跌幅上限
  - 建议搜索范围：`0.06 ~ 0.18`
  - 建议步长：`0.02`

### 4.5 板块过滤

- 参数：`buy_sector_rank_top_pct`
  - 默认值：`0.50`
  - 含义：所属板块 `5` 日涨幅排名必须位于前 `50%`
  - 建议搜索范围：`0.20 ~ 0.70`
  - 建议步长：`0.05`

### 4.6 成交额过滤

- 参数：`buy_amount_min`
- 默认值：`250000000`
- 含义：近 `3` 日平均成交额 `>= 2.5 亿`
- 建议搜索范围：`1.0亿 ~ 8.0亿`
- 建议步长：`0.5亿`

## 5. 卖出规则

### 5.1 分级止盈减仓 `sell_trim`

- 参数：`sell_trim_profit_threshold`
  - 默认值：`0.08`
  - 含义：持仓盈利达到该值后，若出现放量长上影则减仓
  - 建议搜索范围：`0.05 ~ 0.15`
  - 建议步长：`0.01`

- 参数：`sell_trim_upper_shadow_ratio`
  - 默认值：`0.03`
  - 含义：长上影比例阈值
  - 建议搜索范围：`0.02 ~ 0.08`
  - 建议步长：`0.01`

- 参数：`sell_trim_volume_multiple`
  - 默认值：`1.2`
  - 含义：放量倍数阈值
  - 建议搜索范围：`1.0 ~ 2.0`
  - 建议步长：`0.1`

- 参数：`sell_trim_fraction`
  - 默认值：`0.5`
  - 含义：减仓比例
  - 建议搜索范围：`0.25 ~ 0.75`
  - 建议步长：`0.25`

### 5.2 跌破 MA5 放量清仓 `sell_break_ma5`

- 参数：`sell_break_ma5_volume_multiple`
  - 默认值：`1.0`
  - 含义：跌破 `MA5` 时需要放量到该倍数以上才清仓
  - 建议搜索范围：`0.8 ~ 1.8`
  - 建议步长：`0.1`

### 5.3 动态移动止盈 `sell_drawdown`

第一档：
- 参数：`sell_drawdown_profit_threshold`
  - 默认值：`0.10`
  - 建议搜索范围：`0.06 ~ 0.15`
  - 建议步长：`0.01`
- 参数：`sell_drawdown_threshold`
  - 默认值：`0.05`
  - 建议搜索范围：`0.03 ~ 0.08`
  - 建议步长：`0.01`

第二档：
- 参数：`sell_drawdown_profit_threshold_mid`
  - 默认值：`0.18`
  - 建议搜索范围：`0.12 ~ 0.24`
  - 建议步长：`0.02`
- 参数：`sell_drawdown_threshold_mid`
  - 默认值：`0.06`
  - 建议搜索范围：`0.04 ~ 0.10`
  - 建议步长：`0.01`

第三档：
- 参数：`sell_drawdown_profit_threshold_high`
  - 默认值：`0.30`
  - 建议搜索范围：`0.20 ~ 0.40`
  - 建议步长：`0.05`
- 参数：`sell_drawdown_threshold_high`
  - 默认值：`0.08`
  - 建议搜索范围：`0.06 ~ 0.12`
  - 建议步长：`0.01`

注意：
- 三档阈值建议保持单调：
  - `profit_threshold < profit_threshold_mid < profit_threshold_high`
  - `threshold <= threshold_mid <= threshold_high`

### 5.4 时间止损 `sell_time_stop`

- 参数：`sell_time_stop_days`
  - 默认值：`10`
  - 建议搜索范围：`5 ~ 15`
  - 建议步长：`1`

- 参数：`sell_time_stop_return_threshold`
  - 默认值：`0.02`
  - 建议搜索范围：`0.00 ~ 0.05`
  - 建议步长：`0.01`

固定条件：
- `close <= MA20`

### 5.5 盈转亏卖出 `sell_flip_loss`

- 无单独数值阈值
- 含义：
  - 持仓曾经盈利过
  - 当前收盘价已跌回成本线下方
- 建议搜索方式：
  - 作为开关项 `True / False`

### 5.6 弱市大跌清仓 `sell_market_weak_drop`

- 参数：`sell_market_score_threshold`
  - 默认值：`35`
  - 含义：市场平均分低于该值时进入弱市风控
  - 建议搜索范围：`30 ~ 42`
  - 建议步长：`1`

- 参数：`sell_market_drop_threshold`
  - 默认值：`-3`
  - 含义：个股当日跌幅低于等于该值时清仓
  - 建议搜索范围：`-5.0 ~ -2.0`
  - 建议步长：`0.5`

## 6. 评分相关指标

总分 `100` 分，共 `6` 项。

- `score_ma_trend`
  - 均线多头，权重 `18`
- `score_volume_pattern`
  - 放量上涨 + 缩量回调，权重 `15`
- `score_capital_sector`
  - 资金流入 + 板块强势，权重 `18`
- `score_breakout`
  - 低位启动突破，权重 `22`
- `score_hold`
  - 突破后未破位，权重 `18`
- `score_benchmark`
  - 大盘共振，权重 `10`

## 7. 基础因子指标

回测和评分会用到这些基础指标：

- `MA5 / MA10 / MA20 / MA30 / MA60 / MA120`
- `vol_ma5`
- `ATR14`
- `prior_20_high`
- `CMF(21)`
- `MFI(14)`
- `pct_change`
- `sector_change`
- `sector_up_ratio`
- `benchmark_close`
- `benchmark_ma20`
- `benchmark_prev_ma20`

## 8. 交易成本

- 参数：`fee_rate`
  - 默认值：`0.001`
  - 含义：手续费 `0.1%`
  - 建议搜索范围：`0.0005 ~ 0.0020`
  - 建议步长：`0.0005`

- 参数：`slippage_rate`
  - 默认值：`0.001`
  - 含义：滑点 `0.1%`
  - 建议搜索范围：`0.0005 ~ 0.0030`
  - 建议步长：`0.0005`

## 9. 执行时点

- 信号生成：`T日收盘后`
- 买入执行：`T+1日开盘`
- 卖出执行：触发后 `T+1日开盘`
- 基准指数：`000001`

## 10. 自动化搜索建议

建议把自动化搜索拆成 4 轮：

### 第 1 轮：市场环境
- `market_score_filter_min_avg`
- `market_score_filter_min_ma5`
- `market_require_benchmark_above_ma20`
- `market_require_benchmark_ma20_up`

### 第 2 轮：买入强度
- `buy_strict_score_total`
- `buy_momentum_score_total`
- `buy_min_core_hits`
- `buy_amount_min`

### 第 3 轮：买入过滤
- `buy_low_position_high_ratio_max`
- `buy_20d_gain_max`
- `buy_risk_amplitude_max`
- `buy_risk_max_drop_max`
- `buy_sector_rank_top_pct`

### 第 4 轮：卖出与仓位
- `max_single_position`
- `sell_drawdown_*`
- `sell_time_stop_*`
- `sell_market_score_threshold`
- `sell_market_drop_threshold`

建议输出指标：
- 总收益
- 超额收益
- 最大回撤
- 胜率
- 交易次数
- 平均持仓天数
- 单笔盈亏比

建议先按“超额收益 + 最大回撤 + 交易次数下限”做综合评分，再选最优组合。
