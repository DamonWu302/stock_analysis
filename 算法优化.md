## 优化方向（按优先级排序）

### ✅ 方向一：收紧买入条件，减少无效开仓

当前 `总分≥70` 的阈值**偏低**，导致信号过多、质量参差不齐。

**建议调整：**

1. **提高买入阈值**
   - 增加“必须命中 3 项以上核心指标”
2. **增加“低位”限制**
   - 当前“低位启动突破”权重 16 分，但实际很多买入发生在高位
   - 建议：买入时必须满足 `close ≤ 前20日高点的 95%`，或 **20日涨幅 ≤ 15%**
3. **增加量能约束**
   - 买入前 3 日，不能出现“放量滞涨”
   - 例如：`当日涨幅 < 1% 且 成交量 > 5日均量 1.2 倍` 则禁止买入

------

### ✅ 方向二：重构卖出规则，从“滞后”改为“前瞻”

当前卖出依赖均线死叉，太慢。

**建议改为分级退出机制：**

| 条件                                       | 动作             |
| :----------------------------------------- | :--------------- |
| 持仓盈利 ≥ 8% 且 出现放量长上影 / 高分下降 | 减仓 50%         |
| `close < MA5` 且 成交量 > MA5 量           | 清仓             |
| 最大回撤从买入后高点 > 5%                  | 无条件止损       |
| 持仓 5 天后，若累计收益 < 2%               | 清仓（时间止损） |

这样可以：

- 保住利润
- 防止小亏变深套
- 提高资金周转效率

------

### ✅ 方向三：优化评分权重，降低“追高项”

从你回测看，**“均线多头”和“放量上涨”** 这两项在追高时得分高，但实际风险大。

**建议调整权重：**

- 均线多头：24 → **18**
- 放量上涨+缩量回调：20 → **15**
- 低位启动突破：16 → **22**
- 突破后未破位：12 → **18**

让策略**更偏向低位启动、趋势确认后仍有安全边际**的品种。

------

### ✅ 方向四：增加“市场环境”过滤强度

当前市场过滤只用了“平均分”，但 **平均分本身也是你的策略评分**，容易同源过拟合。

**建议增加外部过滤器：**

1. **大盘趋势过滤**
   - 大盘 MA20 必须向上
   - 大盘收盘价必须在 MA20 上方超过 3 天
2. **涨跌家数过滤**
   - 全市场上涨家数 > 下跌家数
3. **板块强度过滤**
   - 个股所在板块的 5 日涨幅排名前 20%

这些可以**显著降低在市场转弱时的开仓次数**。

------

### ✅ 方向五：引入“仓位管理”而非简单等权

你目前是“剩余现金 / 新开仓数量”等权分配，但：

- 市场好时仓位小
- 市场差时反而容易满仓

**建议改为：**

- 根据**当日总评分**决定总仓位比例
  - 总评分 ≥ 80 → 满仓
  - 总评分 70–80 → 70% 仓位
  - 总评分 < 70 → 不开新仓
- 单只股票最大仓位 ≤ 30%

这样可以**在市场确定性高时多投，不确定性高时少投**。

------

## 三、建议你优先做的 3 件事（快速见效）

1. **提高买入门槛 + 增加低位限制**
   → 减少追高，提高胜率
2. **增加硬止损（-5% 绝对止损）**
   → 避免单笔大亏
3. **增加“止盈线”**
   → 利润达到 8–10% 后，用移动止盈保护
